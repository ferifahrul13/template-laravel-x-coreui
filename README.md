
## About

Template memudahkan dalam membuat aplikasi baru di laravel

## Instalasi

- git clone https://github.com/ferifahrul13/template-laravel-x-coreui.git
- composer install
- cp .env.example .env
- php artisan key:generate
- setting database env
- php artisan migrate
- php artisan db:seed
